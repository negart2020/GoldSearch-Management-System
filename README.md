### ğŸ§  Implemented Search Strategies
This project provides a comparative analysis of four fundamental algorithms as detailed in the technical report:
- **BFS (Breadth-First Search)**: Guaranteed shortest path in unweighted grids.
- **UCS (Uniform Cost Search)**: Optimal search considering movement costs.
- **A* (A-Star)**: Efficient heuristic-based search using Manhattan distance.
- **DFS (Depth-First Search)**: Explores depth first; demonstrated as a non-optimal pathfinding method in the report.

### ğŸ“Š Key Insights from the Technical Report
According to the project documentation, the performance of these algorithms was tested under various scenarios:
1. **Optimality**: BFS, UCS, and A* consistently found the shortest path.
2. **Path Length**: DFS often resulted in significantly longer paths compared to the other three.
3. **Efficiency**: A* showed the most efficient node exploration due to its heuristic nature.
