This project provides a comparative analysis of four fundamental algorithms:
- **BFS (Breadth-First Search)**: Guaranteed shortest path in unweighted grids.
- **UCS (Uniform Cost Search)**: Optimal search considering movement costs.
- **A* (A-Star)**: Heuristic-based search (Manhattan distance) for efficiency.
- **DFS (Depth-First Search)**: Explores depth first; used to demonstrate non-optimal pathfinding.
